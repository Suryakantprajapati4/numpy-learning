# 🔢 NumPy Practice Repository

This repository contains my daily practice and learning of **NumPy**, covering fundamental to intermediate concepts with hands-on examples.

---

## 📌 About

I am learning NumPy step by step by writing and uploading code regularly.
This repository reflects my progress and understanding of array operations in Python.

---

## 📁 Files Included

### 🔹 Basics & Fundamentals

* `numpy_practice.py` – Basic NumPy operations and experiments
* `list_vs_numpy.py` – Comparison between Python lists and NumPy arrays

### 🔹 Array Operations

* `arithmetic.py` – Arithmetic operations on NumPy arrays
* `AggregateFunction.py` – Aggregate functions like sum, mean, std, etc.

### 🔹 Indexing & Slicing

* `array_slicing.py` – Slicing rows, columns, and subarrays
* `3d_array_indexing.py` – Indexing in 3D arrays

### 🔹 Advanced Concepts

* `Broadcasting.py` – Understanding NumPy broadcasting
* `filtring.py` – Filtering data using conditions
* `random_numbers.py` – Generating random integers, floats, shuffle, and choice
